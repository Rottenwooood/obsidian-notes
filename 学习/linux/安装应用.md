## deb
```
# 1
sudo dpkg -i package_name.deb
sudo dpkg -i 
## sudo apt --fix-broken install
# 2
sudo apt install ./package_name.deb
## 注意：文件路径前的 `./` 表示当前目录，不能省略，否则 `apt` 会尝试从软件源查找该包。
# 3
sudo apt install gdebi
sudo gdebi package_name.deb
```
## appimage
理论上可直接运行
```
## 添加权限
chmod +x YourApp.AppImage
./YourApp.AppImage
## 创建快捷方式
创建一个 `.desktop` 文件
[Desktop Entry]
Name=MyApp
Exec=/home/yourname/Apps/YourApp.AppImage
Icon=/home/yourname/Apps/icon.png
Type=Application
Categories=Utility;
Terminal=false
StartupNotify=true
```

```
使用AppImageLauncher图形化安装

sudo add-apt-repository ppa:appimagelauncher-team/stable
sudo apt update
sudo apt install appimagelauncher
```
